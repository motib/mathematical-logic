LearnSAT - Release Notes

Version 1.3.2 (08.04.2013)
  Display antecedent nodes in the assignment tree.
  Fixed bug in display of assignment tree.
  Added display option tree_inc.

Version 1.3.1 (27.03.2013)
  User can specify the order in which variables are assigned.

  Generate dot graphs of assignment tree.
  Code refactoring and cleanup.

  Grid pebbling example added.
  Significantly expanded documentation.

Version 1.1.0 (02.09.2012)
  Significant improvements in software structure and explanations of the algorithms.

Version 1.0.0 (17.04.2012)
  Initial release.
